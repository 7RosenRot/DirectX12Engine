## Peculiarities
- **API** DirectX 12
- **Language** C++20
- **Building System** CMake
- **Application Architecture** The window is implemented through the standard library, graphics are drawn independently within the window

## Launch
- 1. Clone Git repository
- 2. Run `Prewiew.ps1` script
